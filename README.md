# Customer Churn Prediction | Python & ML

## Project Overview
Built a Machine Learning model to predict 
customer churn using Python on Telco dataset.

## Dataset
- Source: Kaggle - Telco Customer Churn
- Records: 7,043 customers
- Features: 21 columns

## Key Results
- Random Forest Model Accuracy: 79%
- Top Churn Factors: MonthlyCharges, 
  tenure, Contract type
- Month-to-month customers churn most

## Steps Performed
- Data Loading & Cleaning
- Exploratory Data Analysis (EDA)
- Data Visualization (Matplotlib, Seaborn)
- Label Encoding
- Train/Test Split (80/20)
- Random Forest Classification Model
- Feature Importance Analysis
